# Wiggly Woosh Landing Page

Welcome to the Wiggly Woosh project! This repository contains the code for a premium yet approachable landing page designed for Wiggly Woosh, incorporating design elements inspired by Vetic and Supertails.

## Project Structure

The project is organized as follows:

```
WigglyWoosh
├── src
│   ├── css
│   │   ├── reset.css          # Resets default browser styles
│   │   ├── variables.css      # CSS variables for colors and typography
│   │   ├── global.css         # Global styles for the landing page
│   │   ├── header.css         # Styles for the header section
│   │   ├── hero.css           # Styles for the hero section
│   │   ├── features.css       # Styles for the features section
│   │   ├── products.css       # Styles for the product spotlight section
│   │   ├── testimonials.css    # Styles for the testimonials section
│   │   ├── how-it-works.css   # Styles for the "How it Works" section
│   │   ├── pricing.css        # Styles for the pricing section (future use)
│   │   ├── faq.css            # Styles for the FAQ section (future use)
│   │   ├── cta.css            # Styles for the call-to-action section
│   │   ├── footer.css         # Styles for the footer section
│   │   └── animations.css      # Styles for animations and transitions
│   ├── js
│   │   ├── main.js            # Main JavaScript entry point
│   │   ├── smooth-scroll.js    # Smooth scrolling behavior
│   │   ├── intersection-observer.js # Intersection Observer API implementation
│   │   ├── accordion.js        # Accordion functionality for FAQs
│   │   ├── carousel.js         # Carousel functionality for images/testimonials
│   │   ├── navbar.js           # Navigation bar functionality
│   │   └── lazy-load.js        # Lazy loading for images
│   ├── fonts
│   │   └── fonts.css          # Font styles for the landing page
│   └── sections
│       ├── header.html        # HTML structure for the header section
│       ├── hero.html          # HTML structure for the hero section
│       ├── brand-bar.html     # HTML structure for the social proof brand bar
│       ├── features.html      # HTML structure for the features section
│       ├── products.html      # HTML structure for the product spotlight section
│       ├── how-it-works.html  # HTML structure for the "How it Works" section
│       ├── testimonials.html   # HTML structure for the testimonials section
│       ├── pricing.html       # HTML structure for the pricing section (future use)
│       ├── faq.html           # HTML structure for the FAQ section (future use)
│       ├── cta.html           # HTML structure for the call-to-action section
│       └── footer.html        # HTML structure for the footer section
├── index.html                 # Main HTML file linking all sections
└── README.md                  # Documentation for the project
```

## Setup Instructions

1. **Clone the Repository**: 
   ```bash
   git clone <repository-url>
   cd WigglyWoosh
   ```

2. **Open the Project**: Open the project in your preferred code editor.

3. **Install Dependencies**: If there are any dependencies (e.g., CSS frameworks, JavaScript libraries), install them as per the instructions provided in the respective files.

4. **Run the Project**: Open `index.html` in your web browser to view the landing page.

## Design Guidelines

- The design should reflect a premium yet approachable aesthetic, combining elements from Vetic and Supertails.
- Use the defined CSS variables for consistent styling across the landing page.
- Ensure that the layout is responsive and user-friendly.
